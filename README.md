# Dragon Dungeon RPG

A browser-based role-playing game built with HTML, CSS, and JavaScript. Players explore locations, manage stats and inventory, fight monsters, and ultimately attempt to defeat the dragon.

## Features

- Player stats system:
  - Experience points (XP)
  - Health
  - Gold
- Turn-based combat mechanics with:
  - Multiple enemy types
  - Randomized hit and damage calculations
  - Weapon break chance
- Weapon and inventory system:
  - Weapon upgrades with increasing power
  - Ability to sell weapons
- Multiple game locations:
  - Town square
  - Store
  - Cave
  - Combat scenes
- Final boss battle (dragon)
- Win and lose states with replay functionality
- Hidden easter egg mini-game based on random number selection
- Dynamic UI updates using DOM manipulation

## Usage

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser  
   (or run using Live Server in Visual Studio Code).
3. Use the on-screen buttons to navigate locations, manage resources, and fight monsters.
